# schema-harness.github.io

Homepage for Schema Harness, served via GitHub Pages at
<https://schema-harness.github.io>.

The site is the Schema launch post — **"Frontier Models with Our Harness
Achieve ~99% on ARC-AGI-3 Public"** — a single self-contained `index.html`.

Plain static HTML — push to `main` to deploy.
The `.nojekyll` file disables Jekyll processing so files are served as-is.
